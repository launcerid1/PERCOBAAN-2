## 🎮 Controls

| Key | Action |
|-----|--------|
| ⬅️ Left Arrow | Move Left |
| ➡️ Right Arrow | Move Right |
| ⬆️ Up Arrow / Space | Jump |
| 🔄 R | Restart Game |

---

## ✨ Features

- 🏃 Smooth side-scrolling platformer movement
- 🪙 Coin collection system
- 🧱 Brick blocks and ground collision
- 🚩 Flagpole at the end of the level
- 💀 Game Over & Win screens
- 🎨 Pure Canvas rendering — no external assets
- ⚡ Lightweight and fast-loading

## 🛠️ Tech Stack

🖥️ HTML5
🧠 JavaScript (Vanilla)
🎨 HTML5 Canvas API
