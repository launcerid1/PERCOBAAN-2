## 🕹️ Kontrol

| Tombol | Aksi |
|---|---|
| `←` | Bergerak ke kiri |
| `→` | Bergerak ke kanan |
| `Spasi` | Lompat |
| `Enter` | Mulai / Jeda |

---

## ✨ Fitur

- 🍄 Gerakan klasik Mario: berjalan, lompat, jatuh
- 🏰 Side-scrolling platformer gameplay
- 🪙 Koin yang bisa dikumpulkan dan power-up
- 👾 Musuh AI (Goomba, Koopa Troopa)
- 📈 Level progression dan sistem skor
- 📱 Kontrol responsif

---

## 🛠️ Tech Stack

- **HTML5** — struktur dan elemen game
- **CSS3** — styling dan tampilan
- **JavaScript (Vanilla)** — logika, fisika, dan render game

---

## 🚀 Cara Menjalankan Secara Lokal

```bash
# 1. Clone repository
git clone https://github.com/launcerid1/PERCOBAAN-2.git

# 2. Masuk ke direktori
cd PERCOBAAN-2

# 3. Buka index.html di browser
```

Atau gunakan ekstensi **Live Server** di VS Code untuk pengalaman development lebih nyaman.
