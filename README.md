# Super Mario Bros Clone 🎮

A simple Super Mario Bros-style game built with HTML, CSS, and JavaScript.

![Mario Running](https://media.tenor.com/1YjMIxwR3B4AAAAi/mario.gif)

## 🎮 Live Demo

Play the game here: [https://launcerid1.github.io/PERCOBAAN-2/](https://launcerid1.github.io/PERCOBAAN-2/)

![Game Preview](https://media.tenor.com/1YjMIxwR3B4AAAAi/mario.gif)

## 🕹️ Controls

| Key | Action |
|-----|--------|
| `←` | Move Left |
| `→` | Move Right |
| `Space` | Jump |
| `Enter` | Start / Pause |

## ✨ Features

- Classic Mario movement and jumping
- Side-scrolling platformer gameplay
- Collectible coins and power-ups
- Enemy AI (Goombas, Koopa Troopas)
- Level progression and score system
- Responsive controls

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## 🚀 How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/launcerid1/PERCOBAAN-2.git
   ```
2. Open `index.html` in your browser.
